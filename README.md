# Intelligent Resume Screening System

An end-to-end NLP-based resume screening and ranking system built using Python, BERT embeddings, and machine learning.

## Problem
Companies receive thousands of resumes per role. Manual screening is slow, inconsistent, and biased.

## Solution
This system:
- Processes structured resume data
- Generates semantic embeddings using BERT
- Ranks candidates against a job description
- Classifies IT vs Non-IT profiles

## Tech Stack
Python, pandas, scikit-learn, NLTK, Sentence-Transformers

## Project Structure
data/ → resume dataset
src/ → production code
notebooks/ → exploration
